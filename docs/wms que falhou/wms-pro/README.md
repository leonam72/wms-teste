# WMS Pro

Sistema de gestão de armazém — FastAPI + SQLite/PostgreSQL · HTML/CSS/JS modularizado.

## Documentação

Toda a documentação está em [`docs/`](docs/).

**Ponto de entrada para IAs e desenvolvedores:** [`docs/AI-START-HERE.md`](docs/AI-START-HERE.md)

## Setup rápido

```bash
# Linux/macOS
chmod +x setup.sh && ./setup.sh

# Windows
setup_windows.bat
```

Login padrão após setup: `admin` / `Admin@123`  — troque a senha no primeiro login em `http://localhost:8000`
