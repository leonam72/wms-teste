# Planta Baixa

## Objetivo

Visualizar e editar o layout do depósito.

## Escopo

Descreve a tela de planta baixa identificada no inventário.

## Conteúdo

## Como é acessada

Botão `data-page="floorplan"` no painel principal.

## Campos

- `#page-floorplan`
- modal `fp`

## Ações

- visualizar planta
- abrir modal de edição de elementos

## Validações

- estado da planta carregado

## Estados da tela

- visualização da planta
- modal de edição aberto

## Dependências

- módulo de planta baixa

## Lacunas

- Ferramentas exatas de edição do layout não foram identificadas no código atual.
