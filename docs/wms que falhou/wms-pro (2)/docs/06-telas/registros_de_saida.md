# Registros de Saída

## Objetivo

Consultar e editar registros de saída já persistidos.

## Escopo

Descreve a tela de registros de saída identificada no inventário.

## Conteúdo

## Como é acessada

Botão `data-page="outbound"` no painel principal.

## Campos

- `#outbound-filter-search`
- `#outbound-filter-type`
- `#outbound-filter-status`
- `#outbound-record-list`
- `#outbound-edit-modal`

## Ações

- filtrar registros
- visualizar registros
- abrir modal de edição

## Validações

- estado de registros de saída carregado

## Estados da tela

- listagem de registros
- filtros aplicados
- modal de edição aberto

## Dependências

- módulo de saídas

## Lacunas

- Campos editáveis e regras de alteração dos registros não foram identificados no código atual.
